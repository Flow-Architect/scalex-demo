# ROADMAP - ScaleX

ScaleX is a governed execution layer for revenue-backed client operations.

## Current Hackathon Release

The current release demonstrates one synthetic operation:

- Northstar Dental Group
- Client Implementation Launch
- $8,500 revenue
- $3,935 prepared/approved delivery cost basis after governed run
- $3,200 blocked risk
- $4,565 protected profit
- 53.7% protected margin
- 50.0% margin floor

## Implemented

- Judge Demo Mode with deterministic/local planning proof.
- FastAPI backend with SQLite evidence ledger.
- Vite React dark enterprise control-room UI.
- Hermes planning adapter boundary with Nemotron 3 Ultra-capable attribution.
- Stripe sandbox/test-double finance proof and real test-mode path when configured safely.
- Local policy engine for vendor, spend, revenue, margin, live-money, and evidence checks.
- Optional NeMo Guardrails adapter probing through a configured local Python runtime.
- Optional NemoHermes/NemoClaw planning route behind explicit local configuration.
- Business intake, document review, workforce/job costing, cost basis, blocked risk, evidence
  ledger, and protected profit report.

## Not Implemented In This Release

- Live-money Stripe execution.
- Verified Live Mode.
- Telegram approval flow.
- MCP server or external agent access.
- Production Hermes.
- Production customer workflows.
- Payroll, HR compliance, tax processing, healthcare compliance, or HIPAA support.
- Docker/NemoClaw command execution from this app.
