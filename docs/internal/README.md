# Internal Operator Notes

This directory holds optional Codex/operator handoff notes. These files are not required for
normal local setup, review, or demo recording.

Public users should start with:

- `README.md`
- `START_HERE.md`
- `docs/OPERATOR_GUIDE.md`
- `docs/OPEN_SOURCE_AUDIT.md`

Keep internal notes public-safe: no local machine paths, credentials, private URLs, raw logs,
recordings, uploaded real files, or unverified integration claims.
