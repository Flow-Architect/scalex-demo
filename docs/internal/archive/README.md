# Internal Archive

This directory preserves public-safe historical notes that are useful for future operators but too
verbose or implementation-process-oriented for the main public docs.

Archived files:

- `architecture-build-history.md` - previous detailed architecture notes with build history,
  future MCP notes, and Telegram approval planning.
- `product-spec-build-history.md` - previous product spec with goal-scoped implementation notes
  and deeper backlog context.
- `submission-writeup-build-history.md` - previous submission draft with historical validation and
  milestone notes.
- `demo-script-build-history.md` - previous demo script with recording path history and future
  integration notes.
- `agents-full-operator-notes.md` - previous full operator note retained for historical context.

Do not put secrets, raw logs, uploaded real files, recordings, or private machine paths here.
