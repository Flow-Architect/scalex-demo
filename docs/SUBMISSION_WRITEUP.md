# ScaleX Submission Writeup

ScaleX helps service teams run agent work against paid workflows while protecting gross margin.

ScaleX is a working product-style prototype with a local FastAPI backend, SQLite audit ledger, policy-gated spend, deterministic agent outputs, and a Vite dashboard for the Harbor Fleet Services sample workflow.

The product direction is integration-backed proof: isolated Hermes/GPT-5.5 planning, Stripe test-mode payment objects, and NemoClaw or policy safety governance where safe. Current fallback paths must stay clearly labeled and must not be described as live production integrations.
